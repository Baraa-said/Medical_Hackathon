## ✨ الميزات

### 🚨 نظام الطوارئ
- زر SOS للحصول على مساعدة فورية
- أرقام الطوارئ والخطوط الساخنة
- معلومات مراكز المساعدة القريبة

### 💬 المرشد النفسي الافتراضي
- دردشة تفاعلية مع مرشد نفسي ذكي
- نصائح وإرشادات نفسية
- دعم عاطفي ونفسي للضحايا

### 📝 اختبار المعلومات
- اختبار تفاعلي لقياس الوعي بالتحرش الإلكتروني
- 15 سؤال تثقيفي مع إجابات مفصلة
- نتائج فورية مع توضيحات

### 📚 المحتوى التثقيفي
- مقالات شاملة حول التحرش الإلكتروني
- رسوم توضيحية ومعلومات بصرية
- فيديوهات تعليمية
- نصائح سريعة للحماية

## 🚀 كيفية التشغيل

### الطريقة الأولى: فتح مباشر
1. افتح مجلد المشروع
2. انقر نقراً مزدوجاً على ملف `index.html`
3. سيفتح الموقع في المتصفح الافتراضي

### الطريقة الثانية: خادم محلي
```bash
# إذا كان لديك Python مثبت
cd anti-harassment-website
python -m http.server 8000

# أو إذا كان لديك Node.js
npx http-server

# ثم افتح المتصفح واذهب إلى
# http://localhost:8000
```

### الطريقة الثالثة: Live Server (VS Code)
1. افتح المشروع في VS Code
2. ثبت إضافة "Live Server"
3. انقر بالزر الأيمن على `index.html`
4. اختر "Open with Live Server"

## 📄 الصفحات

### 🏠 الصفحة الرئيسية (`index.html`)
- السؤال الرئيسي: "هل تتعرض لتحرش أو ابتزاز إلكتروني الآن؟"
- أربعة أزرار رئيسية للوصول لجميع الخدمات
- تصميم متجاوب وجذاب

### 🚨 صفحة الطوارئ (`emergency.html`)
- أرقام الطوارئ العامة (شرطة، إسعاف، دفاع مدني)
- خطوط المساعدة المتخصصة
- معلومات مراكز المساعدة القريبة
- نصائح مهمة عند طلب المساعدة

### 💬 صفحة المرشد النفسي (`chatbot.html`)
- واجهة دردشة تفاعلية
- أزرار سريعة للمواضيع الشائعة
- ربط مع ChatGPT API للردود الذكية
- دعم نفسي وإرشادي

### 📝 صفحة الاختبار (`quiz.html`)
- 15 سؤال تثقيفي بنعم/لا
- شريط تقدم وعداد الأسئلة
- نتائج فورية مع تلوين الإجابات
- توضيحات مفصلة لكل سؤال

### 📚 صفحة التثقيف (`education.html`)
- مقالات شاملة حول التحرش الإلكتروني
- رسوم توضيحية ومعلومات بصرية
- فيديوهات تعليمية من YouTube
- نصائح سريعة وموارد إضافية

## 🛠️ التقنيات المستخدمة

- **HTML5**: هيكل الصفحات
- **CSS3**: التصميم والتنسيق
- **JavaScript**: التفاعل والوظائف
- **Google Fonts**: خط Cairo العربي
- **Responsive Design**: تصميم متجاوب
- **ChatGPT API**: المرشد النفسي الذكي

## ⚙️ الإعدادات

### إعداد ChatGPT API
لتفعيل المرشد النفسي الذكي:

1. احصل على مفتاح API من OpenAI
2. افتح ملف `js/chatbot.js`
3. ابحث عن السطر:
   ```javascript
   const API_KEY = 'YOUR_API_KEY_HERE';
   ```
4. استبدل `YOUR_API_KEY_HERE` بمفتاح API الخاص بك

### تخصيص المحتوى
- **أرقام الطوارئ**: عدّل الأرقام في `emergency.html` حسب بلدك
- **مراكز المساعدة**: أضف معلومات المراكز المحلية
- **الأسئلة**: يمكن تعديل أسئلة الاختبار في `js/quiz.js`

## 🔧 استكشاف الأخطاء

- تحقق من مسارات الصور في HTML

**المرشد النفسي لا يعمل:**

**التصميم لا يظهر:**

- تحقق من مسارات CSS في HTML
