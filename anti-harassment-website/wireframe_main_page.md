## تخطيط الصفحة الرئيسية (Wireframe)

```
+-------------------------------------------------------------------+
| [شعار الموقع/اسم الموقع]                                         |
+-------------------------------------------------------------------+
|                                                                   |
|                                                                   |
|           هل تتعرض لتحرش أو ابتزاز إلكتروني الآن؟                 |
|                                                                   |
|                                                                   |
|                  +-----------------------+                        |
|                  |         SOS           |                        |
|                  +-----------------------+                        |
|                                                                   |
|                                                                   |
|           هل تحتاج لمرشد نفسي؟                                   |
|                                                                   |
|                  +-----------------------+                        |
|                  |   تحدث مع مرشد نفسي   |                        |
|                  +-----------------------+                        |
|                                                                   |
|                                                                   |
|                  +-----------------------+                        |
|                  |      اختبر نفسك       |                        |
|                  +-----------------------+                        |
|                                                                   |
|                  +-----------------------+                        |
|                  |      ثقف نفسك         |                        |
|                  +-----------------------+                        |
|                                                                   |
+-------------------------------------------------------------------+
```

**العناصر الرئيسية:**

1.  **الرلاية العلوية (Header):** شعار الموقع أو اسمه.
2.  **السؤال الرئيسي:** نص كبير وواضح يسأل
